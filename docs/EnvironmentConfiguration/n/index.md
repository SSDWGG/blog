# n


[了解如何下载 n](https://www.jianshu.com/p/a927bcecdbc0)


- 首先你需要安装nodejs环境  ： [进入官网傻瓜式安装即可](https://nodejs.org/en)

- 安装 n  ：   ```npm install -g n```

- 查看命令  ：``` n --help```



### 切换node就是这么简单

<!-- v16.18.1 -->

sudo npm install -g n    安装n

sudo n stable 安装最新稳定版 node

sudo n 16.18.1

sudo n 10.16.3

Node 查看版本
